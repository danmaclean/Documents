NAME

Probes - a module of methods for getting information about probes on the AFFY ATH121501 chip

AUTHOR

Dan MacLean (dan.maclean@tsl.ac.uk)

SYNOPSIS

        use ArrayTools::Probes;
        my $probe = '25431_x_at';
        my $annotation = Probes::get_TAIR_annotations($probe);

DESCRIPTION

The Probes module contains routines for getting Arabidopsis affymetrix probe information. The module does not create objects like some of the other modules, rather its methods are accessed in the manner of subroutines in external files.

METHODS

get_TAIR_annotations

Gets a probes annotation from a TAIR description file, currently the May 2007 annotations. We can change this easily if need be.

        $probe = '25431_x_at';
        $annotation = Probes::get_TAIR_annotations($probe);
